### About Me  

- Currently working as **Software Architect @ Rotoclear GmbH** (Industrial camera systems, Heidelberg).  
- Leading projects on **embedded systems (C Pro, C2, C Basic)** and **internal software solutions** like `Etikettendruck`, `InsightBridge`, and `FieldSync`.  
- Focus areas: **Embedded Linux (V4L2, GStreamer)**, **SvelteKit apps**, **PostgreSQL**, and **workflow automation (ERP + Salesforce)**.
